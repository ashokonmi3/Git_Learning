GIT DEMO – git add, git commit, git log

Step 1: Create a project and initialize Git
mkdir demo-git
cd demo-git
git init

Step 2: Create a file
echo "Hello Git" > file.txt

Step 3: Check status
git status

Step 4: Add file to staging
git add file.txt

Step 5: Check status again
git status

Step 6: Commit the file
git commit -m "first commit"

Step 7: Modify the file
echo "Learning Git is easy" >> file.txt

Step 8: Check status
git status

Step 9: Add updated changes
git add file.txt

Step 10: Commit again
git commit -m "updated file with new line"

Step 11: View commit history
git log

Step 12: View short commit history
git log --oneline
