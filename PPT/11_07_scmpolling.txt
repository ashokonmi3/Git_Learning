EXAMPLE 1 – SCM POLLING (AUTO BUILD ON CODE CHANGE)

---

## STEP 1 – CREATE JOB

New Item → scm-poll-job → Freestyle

---

## STEP 2 – CONFIGURE GIT

Repository URL:
https://github.com/<username>/jenkins-demo.git

Branch:
main

---

## STEP 3 – ADD BUILD STEP

type app.txt

---

## STEP 4 – ENABLE SCM POLLING

Build Triggers → Poll SCM ( we need to give the polling time here )

Schedule:

* * * * *


## STEP 5 – TEST FLOW

1. Modify file in GitHub
2. Add new line:
   Auto trigger test
3. Commit

---

## EXPECTED RESULT

Within 1 minute → Jenkins triggers build automatically

Console Output:
Started by SCM change

---

## LEARNING

## Jenkins continuously monitors Git and auto-builds when code changes
